CSS - Cascading Style Sheets

CSS (Cascading Style Sheets) is used to style and format HTML documents. It controls the colors, fonts, layout, and overall look of a webpage. CSS makes webpages more attractive and user-friendly.

Types of CSS

  • Inline CSS → Written inside an HTML tag using the style attribute.
  • Internal CSS → Written inside a <style> tag in the HTML file.
  • External CSS → Written in a separate .css file and linked using <link>.

Common CSS Properties

Property Usage
color Sets the text color
background-color Sets the background color
font-size Changes the size of the text
border Sets border around an element
margin & padding Controls space outside & inside an element

Steps to Apply CSS

  1. Choose the type of CSS (inline, internal, or external).
  2. Write the CSS properties and values.
  3. Save your file.
  4. Open it in a browser to see the styling applied.
