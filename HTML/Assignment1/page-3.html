JavaScript - Programming for the Web

JavaScript is a programming language used to make webpages interactive. While HTML structures the content and CSS styles it, JavaScript adds dynamic behavior like animations, event handling, form validation, and much more.

Where to Write JavaScript

  • Inline JavaScript → inside an HTML tag using the onclick or similar attributes.
  • Internal JavaScript → inside a <script> tag in the HTML file.
  • External JavaScript → in a separate .js file linked with <script src="...">.

Common JavaScript Functions

Function / Feature Usage
alert() Displays a message in a popup box
document.write() Writes content directly to the page
getElementById() Accesses an HTML element by its ID
console.log() Logs messages to the browser console
Events (onclick, onmouseover) Perform actions when users interact with elements

Steps to Use JavaScript

  1. Choose inline, internal, or external JavaScript.
  2. Write functions or scripts inside <script> tags or .js file.
  3. Connect JavaScript with HTML elements (using IDs, classes, or events).
  4. Run your webpage in a browser to see it in action.
